CSS


Sumber : Wikipedia


    Apa itu CSS?

  1. CSS adalah singkatan dari Cascading Style Sheets.
  2. Ini adalah bahasa stylesheet yang digunakan untuk menggambarkan tampilan halaman web, termasuk tata letak, warna, dan huruf.

    Konsep Utama:

  1. Selektor: Digunakan untuk menargetkan elemen HTML untuk diberi gaya.
  2. Properti: Mendefinisikan aspek gaya, seperti warna, ukuran font, dan margin.
  3. Nilai: Menentukan pengaturan properti, seperti 'merah' untuk warna atau '20px' untuk ukuran font.

    Aturan CSS:

  1. Aturan CSS terdiri dari selektor dan blok deklarasi.
  2. Blok deklarasi berisi satu atau lebih pasangan properti-nilai.
  3. Contoh: h1 { color: biru; ukuran-font: 24px; }

    Spesifisitas:

  1. Menentukan aturan CSS mana yang memiliki prioritas ketika beberapa aturan menargetkan elemen yang sama.
  2. Spesifisitas didasarkan pada kompleksitas selektor dan urutan penampilan.

    Model Kotak:

  1. Menggambarkan cara konten, padding, batas, dan margin elemen dikelompokkan.
  2. Penting untuk mengendalikan tata letak dan jarak.

    Desain Responsif:

  1. CSS memungkinkan pembuatan tata letak responsif yang beradaptasi dengan berbagai ukuran layar menggunakan kueri media.

    CSS Eksternal, Internal, dan Inline:

  1. CSS eksternal disimpan dalam file .css terpisah.
  2. CSS internal didefinisikan dalam tag <style> dalam dokumen HTML.
  3. CSS inline diterapkan langsung pada elemen HTML individu dengan atribut 'style'.
